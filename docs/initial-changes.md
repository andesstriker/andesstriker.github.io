# Initial Website Specification

## Overview
Personal website for Tomas Varon Saldarriaga, deployed via GitHub Pages at `https://andesstriker.github.io`.

## Tech Stack
- Plain HTML, CSS, vanilla JS (no frameworks, no build tools)
- `index.html` + `style.css` + `script.js`
- GitHub Pages deployment (push to `main` branch)

## Design
- **Style:** Minimal & clean — generous whitespace, Inter font, neutral palette
- **Layout:** Single-page, vertically scrolling sections
- **Responsive:** Mobile-first, works on all screen sizes

## Sections
1. **Hero / About Me** — Name, tagline (CosmosDB SDK Team), bio, photo placeholder
2. **Projects / Work** — Links to work GitHub (tvaron3) and personal GitHub (andesstriker)
3. **Interests** — Cybersecurity and AI
4. **Contact / Links** — Email, LinkedIn, both GitHub profiles
5. **Footer** — "Built with Claude 😏"
